Run WebPivotTable as independent application

STEP 1

To run WebPivotTable as an independent application on your machine, you need first install Node.js.  Please follow the instructions on the  Node.js official website to install it.

To check whether Node.js platform has been installed on your machine or not, open a cmd window (for Windows system, if you don't know how to do it, click here ) or a terminal (for Linux/Unix/OS-X systems) and run "node --version" command like:

c:\>node --version

If Node.js installed correctly, you will see Node.js version information.


STEP 2

We assume that you have gotten the WebPivotTable package and unzip it to a local directory, like c:\webpivottable .

In the same cmd window or terminal. Change to WebPivotTable installed directory, and run "npm install" command like:

c:\>cd c:\webpivottable

c:\webpivottable>npm install

You will see some Node.js modules downloaded to your machine.


STEP 3

Run "node app" command to start the server:

c:\webpivottable>node app

If you see messages like “Express server listening on port 8002″, then you have successfully run the WebPivotTable application on your local machine!


STEP 4

Keeping this cmd window or terminal opened, open any browser in your machine and type “http://localhost:8002” into the address box and you will see WebPivotTable application running in your browser.

If your machine is in a LAN/Intranet network, people in the same network can access this WebPivotTable application by open their browsers and link to "http://(IP address):8002" , here (IP address) should be replaced by real IP address of your machine, for example, if IP address of your machine is 192.168.0.100, then the link should be "http://192.168.0.100:8002".
